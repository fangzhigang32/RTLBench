Design a smart curtain controller module with a 1-bit clock input clk, reset rst, 1-bit light sensor input light, 1-bit manual switch input manual, and outputs a 1-bit open signal open and a 1-bit close signal close. When light=0 or manual=1, open is active; when light=1, close is active. module_name is "smart_curtain_controller". input pin: clk, rst, light, manual. output pin: open, close.
